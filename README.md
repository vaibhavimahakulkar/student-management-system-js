# Student Management System

A console-based Student Management System built using JavaScript.

## Features
- Add Student
- View All Students
- Search Student
- Update Student
- Delete Student
- Show Topper

## Concepts Used
- Variables
- Functions
- Arrays
- Objects
- Loops
- Switch Statements
- Linear Search
- Array Manipulation (splice)

## Future Improvements
- Browser UI using DOM
- Data persistence with Local Storage
- Sorting students
- Filter students
